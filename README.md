# QuakeAssets Archive

Curated Quake 1 / NetQuake asset archives used by NexQuake quickstart catalogs.

## Layout

- `q1/`: Quake 1 (NetQuake) archives

## Archive Index

| File | Description |
|------|-------------|
| `q1/quake106.zip` | Official Quake 1.06 shareware archive. Used to extract shareware `pak0.pak`. |
| `q1/lq-pak1.zip` | NexQuake-packaged LibreQuake `pak1.pak`. Provides a free/open replacement for registered `pak1.pak`. |
| `q1/farena12.zip` | Rocket Arena v1.2 main game assets (`pak0.pak`). |
| `q1/fasrv12.zip` | Rocket Arena v1.2 server package (`progs.dat`, cfgs, and source zip files). |
| `q1/3wctfc.zip` | 3Wave CTF 4.0 full content pack (`pak0.pak` and `pak1.pak`) for client/common assets. |
| `q1/3wave421d.zip` | 3Wave CTF 4.21d server/progs package (`progs.dat`, `qwprogs.dat`, QC source tree). |
| `q1/3wave251.zip` | Original 3Wave CTF 2.51 package with `progs.dat`, cfg, docs, and many `.ent` overrides. |
| `q1/tf28.zip` | Team Fortress 2.8 distribution (`FORTRESS/` tree with pak/progs/docs/cfg). |
| `q1/fvf42R.zip` | Future vs Fantasy 4.2R mod pack containing multiple PAKs (`pak0`-`pak3`), `progs.dat`, demos, and docs. |

## Notes

These archives are referenced by NexQuake bootstrap catalogs and are expected to remain stable once published.
